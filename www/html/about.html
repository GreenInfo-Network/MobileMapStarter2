About MobileMapStarter

Development of MobileMapStarter is funded by GreenInfo Network.

The Github repository for MobileMapStarter is https://github.com/gregallensworth/MobileMapStarter2/ There, you may find the most recent release, report bugs, request new features, or create and submit pull requests with your suggestions.

MobileMapStarter is based on these techologies:

  • Apache Cordova - The go-to framework for creating hybrid mobile apps.
  • Angular - Programming framework for tying HTML behaviors to data and state changes.
  • Bootstrap - Styling framework for buttons, input widgets, navbars, and more.
  • mobile-angular-ui - A kit which ties together Angular and Bootstrap, and provides additional UI elements and behaviors.
  • Leaflet - The mapping toolkit.
  • leaflet-control-geocoder - The geocoder included in the app.
  • angular-bootstrap-toggle - on Map Settings, that nice toggle control.
  • L.TileLayer.Cordova - Leaflet TileLayer with offline caching to device storage.
  • ESRI Basemap Services - Some of the stock basemaps configured in MobileMapStarter are Powered by Esri.
  • OpenStreetMap - Some of the stock basemaps configured in MobileMapStarter are powered by OpenStreetMap.
